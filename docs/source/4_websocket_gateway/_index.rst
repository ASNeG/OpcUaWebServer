WebSocket Gateway
====================

You can use **WebSocket Gateway** to call OPC UA :term:`Service`\ s from your web application without
any configuration on the Gateway's side. But the JSON API of the gateway assumes that you are familiar with the OPC UA
specifications. In this documentation we're going to learn some basic things to start with the gateway.
